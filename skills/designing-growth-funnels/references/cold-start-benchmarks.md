# Cold-Start Benchmarks

Use this only when the user has no owned metrics or proof. These are planning priors, not market facts, evidence, or promises.

## Rules

- Store them as `benchmark_assumption`, never as source-backed evidence.
- Show them in `final/` as assumptions.
- Do not use them for paid-media scale decisions, revenue forecasts, guarantees, legal/financial claims, or launch-ready exports.
- Replace them with user analytics, CRM data, customer interviews, or current sources as soon as available.

## Default Priors

| Profile | Metric | Planning prior |
| --- | --- | --- |
| SaaS | signup_to_first_value | 2-5% |
| Education / webinar | webinar_or_bot_to_qualified_call | 5-15% |
| Real estate / high ticket | qualified_consultation_show | 10-25% |
| Marketplace | request_to_matched_next_step | 5-15% |
| Local services | lead_to_booked_slot | 10-30% |
| Generic | target_action | 5-15% |

The phrasing should always say "planning prior" or "допущение", not "average market conversion".
